DC全币种套利监控系统 部署说明

====================
快速启动方法：
====================
1. 双击运行 "启动监控系统.bat"
2. 打开浏览器访问：http://localhost:8888
3. 局域网其他电脑访问：http://你的服务器IP:8888
4. 公网访问：需要在路由器做端口映射（8888端口）

====================
功能说明：
====================
1. **界面优化**：全新的现代化界面，支持响应式设计，在不同设备上都能良好显示
2. **真实API数据**：支持从Upbit、Bithumb、Binance、OKX和火币获取实时市场数据
3. **模拟数据优化**：更接近真实市场的价格波动和趋势变化
4. **多交易所支持**：同时监控多个交易所之间的套利机会
5. **飞书通知**：支持通过飞书Webhook推送套利机会通知

====================
使用真实API数据：
====================
1. 在控制面板中找到"真实API数据"开关
2. 开启后，系统会自动从各交易所API获取实时价格数据
3. 首次使用时，可能会出现API请求限制，属于正常现象
4. 如果API连接失败，系统会自动切换回模拟数据模式

====================
部署到公网方法：
====================
方法1：端口映射
- 登录你的路由器后台，找到"端口映射"/"虚拟服务器"设置
- 外部端口：8888，内部端口：8888，内部IP：你的服务器内网IP
- 保存后，访问 http://你的公网IP:8888 即可

方法2：使用内网穿透（推荐，不需要公网IP）
- 使用ngrok、花生壳、frp等内网穿透工具
- 把本地8888端口映射到公网
- 示例（ngrok）：ngrok http 8888

====================
后台运行方法（关闭窗口也能运行）：
====================
1. 按Win+R，输入 shell:startup 打开启动文件夹
2. 把"启动监控系统.bat"的快捷方式放到这个文件夹里
3. 这样服务器开机就会自动启动监控系统

====================
注意事项：
====================
1. 部署到公网后，飞书Webhook的跨域问题会自动解决，不需要再设置浏览器参数
2. 如果需要修改端口，右键编辑bat文件，把最后一行的8888改成你想要的端口
3. 服务器建议24小时开机，就能全天候监控套利机会
4. 使用真实API时，建议设置合理的刷新间隔，避免触发API请求限制
5. 部分交易所API可能会有访问限制，如遇到连接失败请稍后重试

====================
进阶部署（可选）：
====================
如果需要更稳定的服务，可以安装Nginx/Apache等Web服务器：
1. 下载Nginx for Windows
2. 把html文件放到nginx/html目录下
3. 启动nginx，访问 http://localhost 即可
4. 支持配置域名、HTTPS、负载均衡等功能
