# DC套利监控系统 - 部署说明

## 🚀 快速部署（5分钟完成）

### 方式1：本地直接运行
1. 直接打开 `index.html` 文件即可使用
2. 所有功能正常，实盘API自动对接
3. 配置自动保存到浏览器本地存储

### 方式2：部署到Web服务器（公网访问）
#### Windows服务器
1. 把 `index.html` 放到任意目录
2. 打开命令提示符，进入目录，运行：
   ```bash
   python -m http.server 8000
   ```
3. 访问 `http://你的IP:8000` 即可

#### Linux服务器（Nginx）
1. 安装Nginx：
   ```bash
   sudo apt install nginx
   ```
2. 把 `index.html` 放到 `/var/www/html/` 目录
3. 启动Nginx：
   ```bash
   sudo systemctl start nginx
   ```
4. 访问服务器IP即可

#### 免费托管（GitHub Pages）
1. 创建GitHub公开仓库
2. 上传 `index.html` 到仓库
3. 开启GitHub Pages功能
4. 访问 `https://你的用户名.github.io/仓库名` 即可

## 🔧 自定义修改

### 修改UI样式
所有CSS都在HTML文件的 `<style>` 标签内，可以直接修改：
- 修改颜色：修改 `background`、`color` 属性
- 修改布局：修改 `grid`、`flex` 相关属性
- 修改Logo：替换DC Logo的HTML代码

### 修改功能逻辑
所有JS代码都在HTML文件的 `<script>` 标签内：
- 修改刷新频率：修改 `config.refreshInterval`
- 修改利润阈值：修改 `config.profitThreshold`
- 添加新交易所：在 `fetchRealPrices` 函数中添加新的API请求
- 添加新通知渠道：在 `sendFeishuNotification` 基础上添加企业微信、钉钉等通知

### 对接交易API实现自动交易
1. 获取交易所API Key和Secret
2. 添加下单接口调用代码：
   ```javascript
   // 币安下单示例
   async function placeOrder(symbol, side, quantity, price) {
       const apiKey = '你的API Key';
       const secret = '你的API Secret';
       
       const params = {
           symbol: symbol + 'USDT',
           side: side,
           type: 'LIMIT',
           timeInForce: 'GTC',
           quantity: quantity,
           price: price,
           timestamp: Date.now()
       };
       
       // 签名请求...
       const res = await fetch('https://api.binance.com/api/v3/order', {
           method: 'POST',
           headers: { 'X-MBX-APIKEY': apiKey },
           body: new URLSearchParams(params)
       });
       
       return await res.json();
   }
   ```

## 📱 移动端适配
系统已经响应式设计，支持手机、平板直接访问，不需要额外修改。

## 🔒 安全建议
1. 如果部署到公网，建议添加密码保护
2. 交易API密钥不要硬编码到前端代码中，建议使用后端代理
3. 重要数据加密存储
4. 开启Web应用防火墙（WAF）防止恶意访问
