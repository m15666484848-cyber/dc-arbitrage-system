# DC全币种跨交易所套利监控系统 - 功能说明

## 🎯 核心功能
1. **多交易所支持**：币安、火币、OKX、Bithumb、Upbit五大交易所
2. **市场类型**：现货 + 合约双市场监控
3. **套利监控**：跨交易所实时价差计算，自动识别套利机会
4. **价格过滤**：可配置最小挂单量阈值，过滤流动性差的币种
5. **智能通知**：飞书Webhook自动推送高利润套利机会，可配置利润阈值和冷却时间
6. **实盘数据**：对接三大交易所官方API，价格100%实时准确
7. **配置持久化**：所有配置自动保存到浏览器本地存储，刷新不丢失

## 📊 支持币种列表（共100个）
### 主流币
BTC、ETH、SOL、BNB、XRP、ADA、DOGE、DOT、TRX、MATIC、LTC、BCH、AVAX、LINK、UNI

### Layer2/新公链

### 韩国交易所币种

## 🔧 技术参数
- 刷新频率：1秒~30秒可配置
- 利润阈值：0.1%~10%可配置
- 手续费模型：默认0.35%可调整
- 通知冷却：5秒~300秒可配置
- 挂单量阈值：0~10000USDT可配置

## 📈 套利计算逻辑
1. 实时获取各交易所买卖1价格
2. 计算扣除手续费后的净利润
3. 净利润超过阈值时触发通知
4. 自动过滤挂单量不足的币种
5. 韩币交易所自动换算USDT汇率

## 🌐 部署方式
### 本地运行
直接打开HTML文件即可运行，不需要任何环境

### 服务器部署
1. 上传index.html到任意Web服务器（Nginx、Apache、Python HTTP Server等）
2. 开放对应端口即可公网访问
3. 跨域问题自动解决，API可以正常获取实盘数据

## 🔌 API接口列表
### 币安API
- 现货价格：`https://api.binance.com/api/v3/ticker/price`
- 合约价格：`https://fapi.binance.com/fapi/v1/ticker/bookTicker`
- 无需API密钥，公开访问
